03 — הכנה לפני השיעור

5 שלבים.

עשו את זה לפני שמגיעים. כל שלב עם הסבר מה לחכות לראות. אם משהו לא עובד — כתבו ל-Migo.

01
התקינו Node.js
תוכנת בסיס שבלעדיה Claude Code לא יכול לרוץ. כמו Java ל-Minecraft.
 Mac — בטרמינל (⌘+Space → Terminal):
$ curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.7/install.sh | bash

סגרו טרמינל, פתחו מחדש, ואז:

$ nvm install --lts
⊞ Windows — קודם התקינו WSL:
⚠ חשוב: Claude Code לא עובד ב-Command Prompt הרגיל. צריך WSL — טרמינל לינוקס מובנה ב-Windows 10/11.

פתחו Command Prompt כמנהל מערכת (לחצו ימני → Run as administrator):

> wsl --install

הפעילו מחדש, פתחו WSL, ואז הריצו את אותן פקודות כמו Mac (curl + nvm install).

✓ בדיקה: הקלידו node -v — אם רואים מספר, הצליח

03
התקינו Claude Code
פקודה אחת — וזהו. Mac: Terminal. Windows: WSL (לא cmd, לא PowerShell).
$ npm install -g @anthropic-ai/claude-code
✓ איך יודעים שהצליח? הסמן מהבהב שוב ואין טקסט אדום. הודעות "npm update" או "funding" הן לא שגיאות — התעלמו מהן.
